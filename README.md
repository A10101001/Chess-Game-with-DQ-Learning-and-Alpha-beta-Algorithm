# Chess-Game-with-DQ-Learning-and-Alpha-beta-Algorithm
Program to play Chess against DQ Learning and the Alpha-Beta algorithm. In the Menu, you can select to train the model, play against DQ Learning, or play against the Alpha-Beta algorithm and DQ Learning together. Once the model is trained, the learned configuration is saved in an external file. Configuration can be reseted from the Menu. Training can be done against itself (DQ ) or Stockfish.
